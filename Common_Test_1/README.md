# Common_Test_1 Documentation

This folder contains the lens classification pipeline and its associated artifacts.

## Files

- **DeepLense_Classification.ipynb**
  - Jupyter notebook implementing the lens classification pipeline.
  - Explains the model, data requirements, and evaluation steps.
  - Produces confusion matrix, ROC curve, and training curves as outputs.
  - To use: Open in Jupyter/Colab, follow the instructions in the first cell, and run all cells.

- **model_weights_test_1.pth**
  - Trained PyTorch model weights for the classification pipeline.
  - Generated by the training process described in the notebook.
  - To use: The notebook loads these weights automatically for inference and evaluation.

## Outputs
- Confusion matrix, ROC plot, and training curves are generated by running the notebook.
- Outputs are displayed inline and can be saved as images if needed.
